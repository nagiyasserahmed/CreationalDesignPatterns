# Creational Design Patterns (C#)

This repository contains simple examples of **Creational Design Patterns** implemented in **C#** using a console application.

## Patterns Covered
- Singleton
- Factory Method
- Abstract Factory
- Builder
- Prototype

## Purpose
The goal of this repository is to:
- Understand when to use each creational pattern
- See clean and minimal C# examples
- Practice object creation principles

## Tech Stack
- C#
- .NET Console Application

## Notes
Each pattern is implemented in a simple and beginner-friendly way, focusing on clarity rather than complexity.
